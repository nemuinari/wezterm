# WezTerm Configuration

Kanagawa テーマベースの WezTerm 設定。PowerShell 7 をデフォルトシェルとして使用。

---

## 環境

| 項目             | 設定値                                   |
| ---------------- | ---------------------------------------- |
| シェル           | PowerShell 7 (`pwsh.exe -NoLogo`)        |
| フォント         | FiraCode Nerd Font Mono / Regular / 20pt |
| カラースキーム   | Kanagawa (Gogh)                          |
| 背景色           | `#0c0c0c`（不透明度 95%）                |
| レンダラー       | OpenGL                                   |
| スクロールバック | 5,000 行                                 |

---

## キーバインド

### ペイン操作

| キー             | 動作                                 |
| ---------------- | ------------------------------------ |
| `Alt + Enter`    | 下方向へ分割（高さ 35%）             |
| `Alt + S`        | 右方向へ分割（幅 50%）               |
| `Ctrl + Alt + W` | ペインナビゲーションモードへ切り替え |

### タブ・ウィンドウ

| キー      | 動作                 |
| --------- | -------------------- |
| `Alt + T` | 新規タブを開く       |
| `Alt + W` | 新規ウィンドウを開く |

### ペインリサイズ

| キー      | 動作                   |
| --------- | ---------------------- |
| `Alt + H` | 左へリサイズ（5 セル） |
| `Alt + J` | 下へリサイズ（2 セル） |
| `Alt + K` | 上へリサイズ（2 セル） |
| `Alt + L` | 右へリサイズ（5 セル） |

### スクロール

| キー               | 動作                   |
| ------------------ | ---------------------- |
| `Shift + ↑`        | 1 行上へスクロール     |
| `Shift + ↓`        | 1 行下へスクロール     |
| `Shift + PageUp`   | 1 ページ上へスクロール |
| `Shift + PageDown` | 1 ページ下へスクロール |

### クリップボード・コピーモード

| キー                   | 動作                                 |
| ---------------------- | ------------------------------------ |
| `Ctrl + V`             | クリップボードから貼り付け           |
| `右クリック`           | クリップボードから貼り付け（マウス） |
| `Ctrl + Shift + Space` | コピーモードへ入る                   |

---

## キーテーブル

### ペインナビゲーション（`Ctrl + Alt + W` で起動）

`Ctrl + Alt + W` を押した後、以下のキーが有効になります（ワンショット）。

| キー     | 動作                 |
| -------- | -------------------- |
| `H`      | 左のペインへ移動     |
| `J`      | 下のペインへ移動     |
| `K`      | 上のペインへ移動     |
| `L`      | 右のペインへ移動     |
| `Escape` | キーテーブルを抜ける |

### コピーモード（`Ctrl + Shift + Space` で起動）

コピーモード中は以下のキーが使えます。

| キー            | 動作                           |
| --------------- | ------------------------------ |
| `H / J / K / L` | カーソル移動（左/下/上/右）    |
| `V`             | セル選択モードへ切り替え       |
| `Y`             | 選択範囲をコピーしてモード終了 |
| `Escape` / `Q`  | コピーモードを終了             |

---

## タブバー

カスタムの `format-tab-title` イベントによりタブの見た目を制御。

| 状態         | 背景色    | 文字色    |
| ------------ | --------- | --------- |
| アクティブ   | `#2a2a37` | `#dcd7ba` |
| 非アクティブ | `#1f1f28` | `#727169` |

- ファンシータブバー・新規タブボタンは非表示
- タイトルバーはウィンドウと同色（`none`）でシームレスに統合
